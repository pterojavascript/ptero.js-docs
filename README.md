# Ptero.js - Pterodactyl Wrapper for Node.js
## <ins>Quick Start</ins>

### Javascript
```javascript
const pterojs = require("ptero.js");
const client = new pterojs.Client("host", "token");
```

### Typescript
```typescript
import pterojs from "ptero.js";
const client = new pterojs.Client("host", "token");
```
